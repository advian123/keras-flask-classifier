# Keras & Flask Classifier Web App

## Usage
1. Clone this repository
2. `keras-flask-classifier`
3. `python main.py`
4. Open `http://127.0.0.1:5000/`
5. 

```
git clone 
cd keras-flask-classifier
python main.py

open http://127.0.0.1:5000/ on your browser
click the image upload button
```
